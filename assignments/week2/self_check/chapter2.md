## Self-Check 5
### Trace the evaluation of the following expressions, and give their resulting values
```java
System.out.println(2 + 2 + 3 + 4);
```
> a. 11  
> - 2 + 2 = 4  
> - 4 + 3 = 7  
> - 7 + 4 = 11  
```java
System.out.println("2 + 2" + 3 + 4);
```
> b. "2 + 234"  
> - "2 + 2" + 3 = "2 + 23"  
> - "2 + 23" + 4 = "2 + 234"  
```java
System.out.println(2 + " 2 + 3 " + 4);
```
> c. "2 2 + 3 4"  
> - 2 + " 2 + 3 " = "2 2 + 3 "  
> - "2 2 + 3 " + 4 = "2 2 + 3 4"  
```java
System.out.println(3 + 4 + " 2 + 2");
```
> d. "7 2 + 2"  
> - 3 + 4 = 7  
> - 7 + " 2 + 2" = "7 2 + 2"  
```java
System.out.println("2 + 2 " + (3 + 4));
```
> e. "2 + 2 7"  
> - 3 + 4 = 7  
> - "2 + 2 " + 7 = "2 + 2 7"

```java
System.out.println("(2 + 2) " + (3 + 4));
```
> f. "(2 + 2) 7"  
> - 3 + 4 = 7  
> - "(2 + 2) " + 7 = "(2 + 2) 7"  
```java
System.out.println("hello 34 " + 2 * 4);
```
> g. "hello 34 8"  
> - 2 * 4 = 8  
> - "hello 34" + 8 = "hello 34 8"

> h. "2(int) 2.042"
